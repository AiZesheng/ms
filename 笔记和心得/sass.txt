1.@mixin就相当于存一段css，完了在别的文件中调用
_mixin.scss里写：
// block得有宽度margin左右为auto才能居中 
@mixin center-block { 
	margin-left: auto;
 	margin-right: auto; 
}
在其它文件导入mixin
@import 'mixin'; 
#header { 
	width:1000px;
	@include center-block; 
} 
.gallery{ 
	width:600px; 
	@include center-block; 
}

2.@extend:
.a{
	background: yellow;
}
.b{
	color: #fff;
	@extend .a;		// 就相当于 background: yellow;
}

@mixin与@extend的区别：
.button {
	background: yellow;
}

.button1 {
	@extend: .button;
}

.button2 {
	@extend: .button;
}
如果代码像上面那样写
@extend编译成的css是这样的：
.button, .button-1, .button-2 {  
    background: green;  
}
这个就是DRY风格的css，也就是没有重复片段的css
而@mixin编译成的css是这样的：
.button {  
    background-color: green;  
}

.button-1 {  
    background-color: green;  
}

.button-2 {  
    background-color: green;  
}
